# 🗄️ E-commerce Database Analysis
---------------------------
## 🗂️ Project Overview
This project focuses on designing and analyzing an e-commerce database using SQL. Data generated with AI prompt.
It includes database schema creation, data insertion, and analytical queries. The final stage has VIEWS for 5 different stakeholders roles.

<img width="1424" height="888" alt="ER_diagram" src="https://github.com/user-attachments/assets/6a8fbcb7-12a4-4773-9b9e-9c42906f448f" />

## 📗 Features
- Database schema design
- Sample data insertion
- Analytical queries
- Common Table Expressions (CTEs)
- Views for reporting

## 🧑‍💻 Technologies Used
- SQL (CTEs, Views, Joins)
- Relational Database Concepts
- AI prompts for random data values

## 🧬 Project Structure
- `/data`: SQL script with all the inserts , generated by AI
- `/docs`: tha final presentation in pdf file
- `/queries`: SQL queries for Basic analysis, CTEs and views for each stakeholder
- `/schema`: SQL script for database and tables creation

## 🧑🏻‍🎓 What I Learned
- Designing relational schemas
- Writing analytical SQL queries
- Using CTEs and views for cleaner analysis
